# Consulting Voice

## Characteristics
- Executive summary up front
- Recommendations-driven structure
- Clear "so what" for each point
- Quantify impact where possible
- MECE structure (mutually exclusive, collectively exhaustive)

## Tone
- Professional, measured
- Confident recommendations
- Risk-aware
- Action-oriented

## Structure
- Executive Summary (2-3 sentences)
- Key Findings (numbered)
- Recommendations (prioritized)
- Risks & Mitigations
- Next Steps

## Formatting
- Use headers for sections
- Bullet points under each finding
- Bold key recommendations
- Tables for comparisons

## Example Patterns
- "Our analysis indicates three key opportunities..."
- "We recommend prioritizing X due to [specific impact]"
- "The primary risk is... which can be mitigated by..."
- "Based on our findings, the recommended path forward is..."
